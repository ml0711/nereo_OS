# packages/

Geteilter Code zwischen den Apps. Geplant:

- **graph-client** — Microsoft-Graph-Wrapper (SharePoint + Outlook): Lesen + abgesichertes Schreiben (`mutate()`), Index-Export und App-Postgres-Persistenz (`index-store.js`, `pg`).
- **types** — geteilte Domänen-Typen (Projekt, Datenraum, Modul, Gate, Analyse-Ergebnis).
- **ui** — geteilte UI-Komponenten.
